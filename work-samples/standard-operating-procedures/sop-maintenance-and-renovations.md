## Objective:

The objective of this SOP is to detail the procedures for handling maintenance and renovations.

## Procedure:

To handle maintenance and renovation works, follow the below three step process.

* Add Maintenance Call
* Find Vendor for Active Maintenance Calls
* Close Maintenance Call

![](<../../.gitbook/assets/Unknown image (222)>)

### Step 1: Add Maintenance Call

The objective of this step is to get everything done within 1 hour from the time we received request from tenant. Right from scheduling a Vendor to getting it repaired today or tomorrow or no later than 3 days.

### Process New Maintenance Call:

Fill all the following fields:

* In Maintenance Call Name field, enter the problem details and address (example toilet leaking on 162 maint st e hampton). Acquire details precisely what is wrong specifically to accurately describe it to a Vendor. Refer to [Maintenance Call Instructions](https://docs.google.com/spreadsheets/d/10BVo902L1pyc6J_IWc7DqGnQCc3RsVQE_axoylcCNzA/edit#gid=0) for more details.
* From Is this Emergency? drop down list, choose the desired Emergency status (Unknown, Yes, or No)
* From Maintenance Call type of issue drop down list, select the desired Type of issue based on the Tenant’s problem
* Enter any other details about the issue in Maintenance Call other notes to describe incident\* field
* From the dropdown list, choose the desired property
* From Select Tenant Who Called drop down list, select the desired Tenant name. After selecting, click on “ [Click here to go to Tenant Record details in CRM](https://crm.zoho.com/crm/org635091059/tab/Contacts/2135217000196019130)” to know more details about the Tenant.

_Note: If you have not found the Tenant details, select the checkbox next to Check this If you don't have Tenant info._

* In Tenant Phone field, enter Tenant’s Phone number
* In Tenants Detailed Available Date and Time field, enter Tenants detailed available date and time for today, tomorrow, and following day and ask if they work from home. Get full tenant availability today, tomorrow and day after tomorrow. Ask if they work at home. E.g., I’m free today 8 AM to 10 PM, tomorrow 9-4 PM. And day after tomorrow 8-4 PM. I work from home. Confirm number who will be at home.
* In “Next Step to do on maintenance call” field, based on the previous activities, enter what the next steps to be taken.
* From “Next Step Follow up date and Time” field, select the desired date using the date picker
* From “is this for maintenance or renovation/inspection” field, select the desired category (inspeciton, maintenance, Remodelling/flip expense, or other)
* Future Replacement Needed Notes
* Choose the desired category from Vendor Category ex plumber, electrician drop down list
* Attach the File relating to maintenance call
* Fields for the Photo of the Problem
* Photos of problem: Upload relevant pictures that explains the problem. Note: Select checkbox next to “Check if we have photos of problem” if we have relevant pictures.
* Using the date picker, choose the desired Date and Time of Call
* Try to diagnose or troubleshoot. Refer to “ [Click here to check Diagnosis Questions Sheet](https://docs.google.com/spreadsheets/d/1n38BfhXNUQbbTuDZBW6jVjfRb4oV_lio6UfwwZ28Jb0/edit#gid=0)” to see what we can do to diagnose. If a diagnosis type is not available, we must create it the same day with management assistance. If we don’t have diagnosis questions, check if we need to create missing diagnosis questions.

Vendor for Doing Job and details related to Vendor:

* If they are unable to find a Vendor from above step, load other vendors from other submarkets using Zoho Map under “Section for Loading Vendors from Order to Call Module based on Sub Market and Type of Vendor.

_(HINT for Order to Call) - system limitation unable to select multiple submarkets on single vendor; so if we know a specific vendor services in multiple submarkets then clone them and add in a new sub market - screen recording for reference). * If still they are unable to find a Vendor from the above step, load Vendors from Vendors Module under “Section for Loading Vendors from Vendors Module based on Sub Market and Type of Vendor”

_HINT - Vendors found from thumbtack, if they are good ones - Add in Order to call. Click on the Create vendor drop down arrow mark, select add in order to call._

* Load using Zoho Map: Places Search -> it goes to google search and pulls based on search keyword (roof repair) and the location.
