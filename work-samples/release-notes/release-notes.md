Production release on 2 March 2022

Version 1.0

## Purpose

This document provides an overview of the enhancements, bug fixes, and new features that have been added in the new version.

## Scope

This document is intended to help users become familiar with the changes, bug fixes, enhancements, and new features that have been added in the new version.

## New Features

A new feature has been added that allows users to view attrition analysis for developers in the client’s software estate using a pie chart that compares Total BCE and the number of developers.

This chart provides information for the following groups: New, New-Inactive, Long-Term Staff, and Inactive, represented in green, yellow, blue, and grey. The second chart provides a detailed explanation of each group, including the Total BCE value, percentage of Total BCE, and number of developers when a user hovers over any group.

Overall, for Total BCE, the New group made the smallest contribution, while the Long-Term Staff group made the highest contribution. In comparison, for the number of developers, the New-Inactive group made the smallest contribution, while the Long-Term Staff group made the highest contribution.
