# AI Agents

### 🤖 Documentation & AI Agent Experiences

In my work, I’ve designed and built AI agents that directly support documentation and communication workflows. Each agent was created with a clear purpose: to reduce repetitive effort, improve consistency, and make technical information more accessible.

* **Style Guide Assistant**\
  I developed this agent to automatically review documentation against internal style guides. It checks grammar, tone, and formatting, ensuring every document meets organizational standards. This reduced manual review time and helped teams maintain consistency across large sets of content.
* **AI Human Writer**\
  This agent focuses on transforming technical drafts into polished, human-centered communication. It adjusts tone, simplifies complex language, and makes content more engaging for diverse audiences. By bridging technical accuracy with readability, it added real value to client-facing proposals and user guides.
* **AI Automatic Procedure Writer**\
  Built from developer demo recordings, this agent listens to workflows and converts them into clear, step-by-step standard operating procedures. It captures technical details directly from demonstrations, then structures them into usable documentation. This innovation saved time for developers and ensured that processes were documented accurately without extra effort.

***

### ✨ Value Added

Through these projects, I demonstrated how AI can be applied to documentation not just as a tool, but as a **collaborative partner**. The agents I created improved efficiency, reduced errors, and allowed teams to focus more on innovation rather than repetitive writing tasks.
