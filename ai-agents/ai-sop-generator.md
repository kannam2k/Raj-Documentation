# AI SOP Generator

**Version:** 1.0\
**Audience:** Operations teams · Process owners · Technical writers\
**Output:** Structured SOP drafts

***

## What this agent does

The AI SOP Generator turns rough process notes into a clear, usable SOP.

It helps convert tribal knowledge into repeatable instructions. It is useful when a process exists, but the documentation does not.

***

## How it works

### 1. Collects the process context

The agent starts by gathering the basics:

* Process name
* Goal or outcome
* Trigger for the process
* Owner or responsible role
* Tools or systems involved
* Required steps
* Exceptions, approvals, or escalation points

If key details are missing, the agent should ask follow-up questions before drafting.

### 2. Organizes the workflow

The agent groups the information into a logical sequence.

It separates the main path from edge cases, prerequisites, warnings, and approvals.

### 3. Drafts the SOP

The agent writes the SOP in a format that teams can review and use quickly.

A strong draft usually includes:

* Purpose
* Scope
* Roles and responsibilities
* Prerequisites
* Step-by-step procedure
* Exceptions or escalation rules
* Review or approval requirements

### 4. Improves clarity

The agent removes vague language and fills structural gaps.

It should prefer direct steps, specific actions, and role-based instructions.

### 5. Prepares the SOP for review

The final output should be easy to review, edit, and approve.

It should read like an operational document, not a brainstorming note.

***

This agent works best when the user provides:

* A real process, not just a topic
* The current way the team performs the work
* Decision points and exceptions
* Names of tools, forms, or systems
* The intended audience for the SOP

The more concrete the input, the stronger the SOP draft.

***

## Best use cases

Use this agent when you need to:

* Document a manual business process
* Standardize recurring team work
* Capture SME knowledge into a repeatable format
* Create a first SOP draft before stakeholder review
